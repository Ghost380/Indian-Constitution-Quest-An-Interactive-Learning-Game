## ğŸŒŸ Overview

Indian Constitution Quest is an interactive educational game designed to teach players about the Indian Constitution through engaging gameplay. Instead of traditional rote learning, this game presents constitutional concepts through multiple-choice questions, progress tracking, and interactive feedback.

**Learning Objectives:**
- Understand fundamental rights and duties
- Learn about the structure of Indian government
- Explore directive principles of state policy
- Discover constitutional history and amendments
- Appreciate the values enshrined in the Constitution

## ğŸ® How to Play

### Starting the Game
1. Open the game in your web browser
2. Click "Start the Quest" button
3. Read the first question carefully

### Answering Questions
1. Select one of the four answer options
2. Click "Submit Answer" to confirm
3. Read the explanation to learn more
4. Click "Next Question" to continue

### Game Controls
- **Mouse/Touch**: Select answer options
- **Hint Button**: Get a helpful constitutional fact (costs 25 points)
- **Submit Button**: Submit selected answer
- **Next Button**: Proceed to next question/result

### Scoring System
- **Correct Answer**: +100 points
- **Using Hint**: -25 points
- **Level Up**: Every 300 points
- **Badges**: Awarded at 40%, 60%, 80% scores

1. Create a new GitHub repository
2. Upload all project files to the repository
3. Go to repository **Settings** > **Pages**
4. Under **Source**, select **Deploy from a branch**

7. Your game will be live at: `https://your-username.github.io/repository-name`
